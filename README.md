# Salah Al-maghrby Portfolio

A modern, responsive portfolio website for Salah Al-maghrby, a software developer with 4+ years of experience in web and app development. This site showcases projects, skills, education, and provides contact information for collaboration and opportunities.

## Features
- **Hero Section**: Introduction and quick links to GitHub, LinkedIn, Email, and Resume.
- **Featured Projects**: Highlights recent projects with live demos and source code links.
- **Skills & Education**: Overview of technical skills and educational background.
- **Contact Section**: Easy ways to get in touch.
- **Dark/Light Theme Toggle**
- **Fully Responsive Design**

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or newer recommended)
- [npm](https://www.npmjs.com/) (comes with Node.js)

### Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/slah0982/my-portfolio.git
   cd my-portfolio
   ```
2. Install dependencies:
   ```sh
   npm install
   ```

### Development
To start the development server:
```sh
npm run dev
```
Visit [http://localhost:8080](http://localhost:8080) in your browser.

### Build
To build for production:
```sh
npm run build
```

### Preview Production Build
```sh
npm run preview
```

### Linting
```sh
npm run lint
```
